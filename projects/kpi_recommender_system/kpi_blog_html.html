Building an Intelligent KPI Recommender System

Published: December 3, 2024 | 16 min read | ML, NLP, Business Analytics

Ask any business analyst: "What KPIs should I track?" and you'll get different answers. With thousands of potential metrics, choosing the right KPIs is surprisingly difficult. I built an intelligent recommender system that solves this using machine learning.

Project Goal: Create an ML platform that recommends the right metrics for any business context using content-based filtering and NLP.

Technology Stack

PythonScikit-learnTF-IDFPandasStreamlitReact

Machine Learning Approach

I implemented content-based filtering using TF-IDF vectorization and cosine similarity to match user context with relevant KPIs.

from sklearn.feature_extraction.text import TfidfVectorizer from sklearn.metrics.pairwise import cosine_similarity # Combine features df['features'] = df['industry'] + ' ' + df['department'] + ' ' + df['goal'] # Create TF-IDF matrix vectorizer = TfidfVectorizer(ngram_range=(1, 2), max_features=100) feature_matrix = vectorizer.fit_transform(df['features']) # Calculate similarity query_vector = vectorizer.transform([user_query]) scores = cosine_similarity(query_vector, feature_matrix)[0]

Results

92% expert approval rate across 20 test scenarios
94% context match rate for top-3 recommendations
3.2x faster than manual KPI research

Key Learnings

  • Domain expertise crucial - spent 40% of time curating KPI database
  • Simple models (TF-IDF + cosine similarity) outperformed complex approaches
  • Explainability builds trust - showing similarity scores was essential
  • Content-based filtering perfect for cold-start problem
